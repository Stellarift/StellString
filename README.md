# StellString
Программа на C++ с меню для выполнения четырех базовых операций со строками: замена пробелов на табуляции, подсчет символов, подсчет слов и проверка на палиндром. 
ПРИМЕЧАНИЕ: в 4 задании вводить палидром нужно всегда с нижнего регистра(маленькой буквы), иначе программа не признает его палиндромом, попытаюсь пофиксить это.
